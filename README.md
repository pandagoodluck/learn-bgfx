## Examples
- example_01_helloworld: 熟悉bgfx初始化、销毁、打印debug信息
- example_02_triangle: 用bgfx绘制三角形，包含初步的shader使用
- example_03_transform: 在example_02_triangle基础上，添加绘制的矩阵参数学习
